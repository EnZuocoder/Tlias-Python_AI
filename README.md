# Tlias智能学习辅助系统 Python后端 AI

### 这次用FastAPI这个Python后端web框架重写了一遍Tlias的后端,用作python的大作业

## 技术栈

### 数据库:万年不变`Mysql`,

### orm框架:`sqlAlchemy` ,orm框架的特点就是不用写sql,不用像java里面额外需要xml文件写sql,直接python程序利用绑定数据库的模型类,操作这些模型类就能实现sql语句一样的效果

### web框架:FastAPI,这个框架有一点比较好,就是有现成的交互式文档来测接口:`/docs`

## 特色:

### AI智能应用:python和AI脱不了关系,在后端搞了个接口,可以连接到deepseek开放的API接口,这样前端那个智能AI助手就活了,就是deepSeek服务器可能接收的请求比较多,经过我后端二道贩子转接一下更慢了,速度反正不理想,有机会搞个蒸馏模型部署到本地玩玩,应该会快点
